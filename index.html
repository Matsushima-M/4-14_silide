1. Claude Codeとは何か

Claude Code = 頭脳 + 装備

頭脳だけ（素のClaude）

  • 会話はできる
  • ファイルは触れない
  • データ分析はできない
  • 調べ物はできない
→

装備あり（Claude Code）

  • 会話もできる
  • ファイルの読み書きができる
  • データ分析・グラフ作成ができる
  • ネット検索もできる

RPGでいうと、素手の勇者に剣と盾を持たせたようなもの
